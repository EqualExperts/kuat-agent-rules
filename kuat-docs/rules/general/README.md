# General Rules

Universal design language and brand guidelines that apply to **all platforms and media types**.

---

## Overview

These rules define the core Equal Experts brand identity and design language. They should be loaded for any creative work, regardless of the output type (slides, web, photography, graphics, etc.).

## How to Use

### For AI Agents

**Always load these files** when working on any Equal Experts creative output.

**Minimum context:** Load the specific file(s) relevant to your task:
- Brand questions → `brand.md`
- Logo placement → `logo.md`
- Writing content → `content.md`
- Color decisions → `colours.md`
- Typography → `typography.md`
- Accessibility → `accessibility.md`

**Full context:** Load all files in this directory (~800 lines).

### For Humans

Read these files to understand the core brand and design language before creating any Equal Experts materials.

## Key Principles

1. **Consistency** - Use the same brand elements across all media
2. **Accessibility** - Ensure all outputs are accessible
3. **Simplicity** - Prefer clean, minimal design
4. **Authenticity** - Represent the real Equal Experts culture
