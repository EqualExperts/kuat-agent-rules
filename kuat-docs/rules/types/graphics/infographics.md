## Overview

These rules govern the creation of infographics - visual representations of information, data, or processes.

---

## Key Principles

1. **Information first** - Design serves the data/information
2. **Clarity** - Easy to understand at a glance
3. **Accuracy** - Data must be represented accurately
4. **Accessibility** - Don't rely on color alone, use labels

---

## Types of Infographics

- **Process flows** - Step-by-step sequences
- **Comparisons** - Side-by-side analysis
- **Timelines** - Chronological information
- **Statistical** - Data-driven visualizations
- **Hierarchical** - Organizational structures

---

## Design Guidelines

### Typography
- Use Lexend font
- Ensure readability at intended size
- Clear hierarchy between headings and body

### Color
- Use brand color palette
- Ensure sufficient contrast
- Use color meaningfully, not decoratively

### Layout
- Clear visual flow
- Adequate white space
- Logical grouping of related information
