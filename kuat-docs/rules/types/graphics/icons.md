## Overview

These rules govern the use and creation of icons in Equal Experts materials.

---

## Key Principles

1. **Recognizable** - Icons should be universally understood
2. **Consistent** - Unified style across icon set
3. **Simple** - Clean lines, minimal detail
4. **Accessible** - Pair with labels where meaning may be unclear

---

## Icon Style

- Line weight: 2px (standard)
- Corner radius: Consistent across set
- Color: Single color, typically from brand palette
- Size: Design at 24x24px, scale as needed
