## Key Principles

1. **Consistency** - Use the same brand elements and terms across all media
2. **Accessibility** - Ensure all outputs are accessible
3. **Simplicity** - Prefer clean, minimal design and clear copy
4. **Authenticity** - Represent the real Equal Experts culture
