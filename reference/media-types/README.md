# Media Types

Per-medium reference. Each medium describes what EE work looks like in that format and owns a `patterns/` subfolder for how specific things are done within it. Media types reference [foundations](../README.md) but never each other.

| Medium | Scope |
|--------|-------|
| [slides/](./slides/) | Presentation decks |
| [web-product/](./web-product/) | Web applications, dashboards, product UI and content |
| [web-marketing/](./web-marketing/) | Marketing websites, landing pages, marketing emails |
| [imagery/](./imagery/) | Photography and graphics (icons, illustrations, infographics) |
| [charts-data/](./charts-data/) | Charts and data visualisation |
