# Slide Patterns

Deck-type patterns — content density, formality, and slide budget differ by deck type.

| Pattern | For |
|---------|-----|

| [case-studies.md](./case-studies.md) | Named client stories (standalone or embedded) |
| [reporting.md](./reporting.md) | Status reports, research findings, audits (read-ahead) |
